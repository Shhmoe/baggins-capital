# Baggins OpenSource Company — Version History

## Current Version: v3.3

### Architecture
- 30 employees across 6 departments
- 5 trading departments (crypto, weather, scalper, sports, avantis leverage)
- 7-gate Banker execution pipeline
- 19-hook Company Clock event system
- Multi-source consensus for weather (7+ APIs)
- Automated pattern analysis and forensic investigation cycles

### Key Capabilities
- Market scanning and classification via Polymarket Gamma API
- Multi-department priority-based capital allocation
- Compliance hard-gating on every bet
- Risk management with statistical circuit breakers
- Adaptive weekly recalibration (CFO, Risk Manager, Compliance)
- Full decision snapshot logging for forensic analysis
- Daily reset cycle with performance archival

### Reference Departments
- **Crypto**: 8-step evaluation (distance, momentum, probability, mispricing, signals, gates, confidence, sizing)
- **Weather**: 7-API consensus with source credibility tracking
- **Scalper**: Multi-timeframe technical analysis on 15-min binary markets
- **Sports**: Bookmaker odds comparison with edge detection
- **Avantis**: Leveraged on-chain trading (Base blockchain)
